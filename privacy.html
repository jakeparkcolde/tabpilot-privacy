TabPilot 개인정보처리방침

최종 업데이트: 2026년 6월 25일

TabPilot은 오래 사용하지 않은 Chrome 탭을 안전하게 휴면 처리하기 위한 브라우저 확장 프로그램입니다.

처리하는 정보

  • 열린 탭의 제목, URL, 도메인
  • 탭의 활성, 고정, 오디오 재생, 로딩, 휴면 상태
  • 탭의 마지막 접근 시각
  • 사용자가 설정한 보호 도메인과 개별 탭 보호 상태
  • 탭 정리 시각, 결과와 재활성화 시각
  • 시스템 전체 및 사용 가능한 메모리 용량

탭 URL과 도메인은 정리 후보를 판정하고 사용자가 목록을 확인하는 기능에만 사용됩니다.

저장과 보관

설정, 보호 도메인과 최근 정리 기록은 Chrome의 로컬 저장소에 저장됩니다. 최근 정리 기록은 최대 100건이며 설정 화면에서 삭제할 수 있습니다. 확장 프로그램을 제거하면 Chrome이 확장 프로그램 저장 데이터를 제거합니다.

외부 전송과 공유

  • 개발자 또는 제3자 서버로 정보를 전송하지 않습니다.
  • 광고, 분석, 추적 또는 사용자 프로파일링을 하지 않습니다.
  • 사용자 정보를 판매하거나 제3자에게 공유하지 않습니다.
  • 원격 코드를 다운로드하거나 실행하지 않습니다.
  • 사람이 사용자의 탭 정보를 열람할 수 없습니다.

권한 사용

  • tabs: 탭 상태 확인, 휴면 처리와 재활성화
  • storage: 설정과 정리 기록의 로컬 저장
  • system.memory: 시스템 메모리 참고 정보 표시
  • alarms: 사용자가 활성화한 자동 정리 주기 실행

Chrome 웹 스토어 제한적 사용 준수

Chrome API를 통해 처리하는 정보는 안전한 탭 휴면 및 메모리 관리라는 단일 목적에만 사용됩니다. 광고, 신용 평가, 판매, 프로파일링 또는 기능과 무관한 목적으로 사용하거나 전송하지 않습니다.
