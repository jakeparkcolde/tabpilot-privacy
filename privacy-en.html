PRIVACY POLICY

TabPilot Privacy Policy

Last updated: June 25, 2026

TabPilot is a browser extension designed to safely put inactive Chrome tabs to sleep.

Information processed

  • Open tab title, URL, and domain
  • Tab active, pinned, audible, loading, and sleeping state
  • Tab last access time
  • User-configured protected domains and per-tab protection state
  • Cleanup time, cleanup result, and reactivation time
  • Total and available system memory capacity

Tab URLs and domains are used only to determine cleanup candidates and show the candidate list to the user.

Storage and retention

Settings, protected domains, and recent cleanup history are stored in Chrome local storage. Recent cleanup history is limited to 100 entries and can be deleted from the settings screen. When the extension is removed, Chrome removes the extension's stored data.

External transfer and sharing

  • No information is sent to the developer or third-party servers.
  • No ads, analytics, tracking, or user profiling are used.
  • User information is not sold or shared with third parties.
  • Remote code is not downloaded or executed.
  • No person can view the user's tab information.

Permission usage

  • tabs: inspect tab state, sleep tabs, and reactivate tabs
  • storage: store settings and cleanup history locally
  • system.memory: display system memory reference information
  • alarms: run user-enabled automatic cleanup checks

Chrome Web Store Limited Use compliance

Information processed through Chrome APIs is used only for the single purpose of safe tab sleeping and memory management. It is not used or transferred for advertising, creditworthiness, sale, profiling, or any purpose unrelated to TabPilot's functionality.

Contact

For privacy and support inquiries, please use the TabPilot support page.
