FAQ

Frequently asked questions

Does sleeping a tab close it?

No. The tab and URL stay in place. When you open the tab again, Chrome reloads the page.

Why are some tabs not shown as cleanup candidates?

Active, pinned, audible, loading, protected tabs, and protected sites are excluded for safety.

When does automatic cleanup run?

Automatic cleanup is off by default. If enabled by the user, it runs at scheduled checks only when available system memory is below the configured threshold.

Can I restore a sleeping tab?

Yes. You can reopen sleeping tabs from the sleeping tab list, and immediately after cleanup you can undo the most recent cleanup.

Are memory savings exact?

Chrome does not provide per-tab memory usage directly to extensions. TabPilot shows an estimate based on changes in system available memory before and after cleanup.

Is tab information sent externally?

No. Settings and cleanup history are stored only in the current device's Chrome storage.

Still having trouble?

Please include the issue description, steps to reproduce, Chrome version, and operating system. If you attach screenshots, hide tab URLs and other personal information.
